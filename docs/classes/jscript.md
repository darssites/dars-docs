# JScript Class

This class is the dars wrapper for the `<script></script>` element of HTML. For more info on JavaScript, go [here](https://www.w3schools.com/js/).

### Syntax

```python
obj = JScript('''
function buttonClick() {
  alert("You have pressed the button!")
}
''')
```

Produces:

```HTML
<script>
function buttonClick() {
  alert("You have pressed the button!")
}
</script>
```
